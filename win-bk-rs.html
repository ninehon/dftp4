系統備份與還原

Windows 安裝完成後，建議要打開系統保護(針對系統碟)。做好各項設置，交機給學員前，再做系統映像備份。

🚩本教學以 Windows 11 (22H2)為本，雖然 Windows 10 也適用，但是部分畫面與功能可能稍有不同。

系統備份使用的是從 Windows 7 以來就存在的備份與還原，系統保護則是使用還原點來保護統變更。這兩個都是內建的功能，不需要另外安裝其他程式即可使用。前者除了建立系統映像，還可以建立系統修復光碟，於系統無法正常開機時使用。後者適用於系統更新、變更，或安裝新的應用程式之後，想恢復到先前狀態時使用。這兩個功能最好結合使用，這樣保護電腦基本上就不需要使用第三方工具。

電腦出現異常想使用系統映像或是還原點進行還原，都要先排除硬體的問題。畢竟軟體上的修復解決不了硬體的問題。還原前也要檢查一下是否有需要另外備份的東西，避免一還原回去卻遺失了其他重要資料或設置。

 

建立備份

操作步驟：

  • 開啟「備份與還原 (Windows 7)」
  • 進行「設定備份」
  • 選擇備份位置與項目
  • 取消備份排程
  • 儲存設定並執行備份

系統映像與還原點差異

系統映像是將整個系統(C 碟)映照為可以用來還原的檔案，一但還原就是整個系統回到製作映像時的狀態。通常是系統凌亂、中毒、常規還原點無法搞定等情形下用來恢復的手段。

而還原點是針對當下的系統狀態所做，對跟系統無關的就不在限制內。例如系統更新、安裝應用程式(安裝需要管理員權限，甚至需要從新開機的那種)等，一般情況下有開啟系統保護的話，系統會自動建立還原點。如果是使用免安裝程式，或僅是文件的變動，是不會觸發生成還原點的。如果對要安裝程式沒把握是否合適，可以在安裝前手動建立還原點，也算一種保護方式。

純粹文件檔案的保護與備份，除了自己可以手動或借助第三方程式來進行之外，Windows 也提供「檔案歷程記錄」的功能，不過由於微軟的政策是希望使用者透過微軟帳戶登入電腦，把檔案備份到 OneDrive 內，所以 Windows 11 的檔案歷程記錄功能居然比 Windows 10 時還精簡(白話就是被閹割了)，雖然還是不錯用，但可由此窺見微軟的居心。

哪個時候適合用還原點？當系統還能正常開機，只不過因為安裝了不適合的軟體無法移除乾淨，或是某個更新安裝之後造成系統異常，更新無法移除或移除不完整，這種情況就很適合利用還原點回到之前狀態。

實際操作上如果還能正常開機，建議到「系統保護」的「系統還原」內操作。因為做了系統映像也會被視為還原點，想回到某個還原點或系統映像都可以從這裡下手，比較省事。

 

用映像檔或還原點修復(正常開機)

如果從新開機進入 WinRE 還是無法還原呢？這時請注意是不是電腦硬體的問題？記憶體、電供、硬碟...等零組件壞了都可能會造成系統不穩定。

 

無法正常開機之處理

當電腦無法常開機的時候，請使用網路組安裝 USB 碟開機進入 Windows 修復環境(RE)。如果手邊剛好沒有開機碟呢？根據微軟的說法，WinRE 會在偵測到下列問題後自動啟動：

  • 連續兩次嘗試啟動 Windows 都失敗
  • 在開機完成後兩分鐘內，連續發生兩次非預期的關機
  • 在開機完成後兩分鐘內，系統連續兩次重新開機
  • 安全開機錯誤 (Bootmgr.efi 的相關問題除外)
  • 只能觸控的裝置上發生 BitLocker 錯誤

後記：系統還原點或系統映像功能已經存在 Windows 多年，可是用第三方軟體來備份的人還是不少。這些軟體賣的價格也沒多便宜，還賣得不錯！這說明了系統內建的還原是陽春了點，功能比不上第三方軟體。另一個原因應該是容錯率，系統內建的說好聽是比較嚴謹，說難聽就是容錯率太低！

會需要還原肯定是電腦有點微恙，一丁點誤差就可能造成還原點甚至系統映像還原失敗... 出現這個的比例高嗎？雖不是很高，但也不少見。所以不能靠著能還原就有恃無恐，雖然如此，整體說來有系統還原功能還是比較有保障，值得採用。
