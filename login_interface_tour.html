登入與介面簡介-德樂道智慧學習管理系統

德樂道智慧學習管理系統有三個介面：管理員 Admin、教師端、學生端。

三種身分登入摘要

  • 管理員 Admin 登入網址：https://admin.deledao.com
  • 教師端登入網址：https://portals.deledao.com/teacher/#/teacher-signin
  • 學生端：使用 Google 瀏覽器登入，新增設定檔(人員)，並開啟同步(必要)。

以下是登入步驟圖解與介面簡介：

 

管理員 Admin 登入

管理員 Admin 登入網址：https://admin.deledao.com

輸入帳號電郵登入，如下圖：

 

管理員 Admin 介面簡介

管理員擁有設置的最高權限，如以下各圖片所示各項功能：

儀表板：檢視各項統計資料與趨勢，查看不同時間內的資訊與報告。

 

學生健康：設置檢測學生健康，並設置發送警報通知對象。查看警報的詳細情況、設置警報多重通知(升級流程)、查看所有報告細節與處理狀況。

 

報告：可根據個別學生、年級或班級、全校等，查詢瀏覽紀錄歷史與處理情況。追蹤以儲存的報告、即時監看指定學生的瀏覽螢幕。

 

政策管理：規範學生瀏覽網路的規則。根據需要設置不同的規則用於不同年級、班級甚至是個人。也能建立範本套用於各個規則。

 

教師端：設置教師端的各項功能，管理教師與學生對應關係，管理學生請求的解封鎖網頁，透過診斷協助教師管理其教師入口網站設定。

 

家長端：設定家長入口網站，管理家長或監護人信箱，自訂邀請電子郵件。

 

配置：系統的全域設定，包含更改密碼，管理用戶(各種身分管理員)，設置身分提供者(如 Google Workspace)，排班(課堂和家長資訊的自動匯入設置)，訂閱報告或警報，產生學生登入二維碼(iPad 適用)。

 

部署：設置 Windows 和 Mac 的部署，根據需要設置排除的網域，設置訪客裝置保護。

教師端介面簡介

解除封鎖請求：這裡可以處理學生請求的解除封鎖，按實際需要予以解除或封鎖。

 

管理課堂：教師在此可以建立新課程、開始上課並進行即時課堂管理、建立課堂規則(建立新網站庫)

 

課堂歷史紀錄：查看課堂的歷史紀錄，觀看課堂擷圖。
