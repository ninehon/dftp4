製作 USB 單一開機工具 - 教學說明

新安裝的電腦至少要有一個能開機的工具，現在很多電腦都沒有光碟機，所以使用 USB 隨身碟就變成首選。

如果手動從頭開始安裝作業系統，可以用微軟官方工具製作安裝 USB 碟，除了安裝 OS，此工具也附有磁碟分割、設置等功能。

把所有開機工具整合到一個隨身碟是不錯的做法，但不是所有電腦都買整合型 USB 的帳，尤其是某些品牌電腦或筆電，對不同方式整合的 USB 支援度也不同。為了避免遇到無法用整合型 USB 開機的尷尬，製作單一開機 USB 是有必要的。如果手上有閒置的舊 USB，只要沒壞，讀寫速度不要慢得離譜即可，不必花錢買新的，用來做這種開機碟最合適了。

這樣的 USB 安裝碟可適用於傳統 BIOS 與 UEFI 的開機模式，在某些特別奇怪的主機板或筆電無法使用 Gparted 時，這個 Windows 10 開機隨身碟還能做磁碟分割，即使要使用還原檔來安裝作業系統，這個原版安裝碟也是第一候補工具。

即使從頭手動安裝，以現在較新的電腦來說，因為有固態硬碟的緣故，使得安裝作業系統的時間大大的縮短。如果只是純安裝作業系統，不含其他應用程式的安裝、設置、更新等，大概不到一小時就可以完成。

當然系統更新與設定，應用程式的安裝、設置等還是要花不少時間。手動安裝與還原安裝各有特點，前者可以打造自己想要的內容，但要花較多時間；後者大幅縮減作業系統與應用程式的建置時間，還原後略加整理就可以有一個適合大部分人的作業系統。請依個別需求做適當的選擇吧！

 

 

製作 Gparted 開機碟

Gparted 是一款免費的磁碟分割區編輯工具，提供圖形介面來進行各種磁碟分割區的操作，能處理包含 Linux、Windows、Mac OS X 等作業系統所支援的磁碟格式，而且可以選擇中文介面。

從 GParted 下載 iso 檔燒錄成可開機光碟，或是做成可開機 USB 碟即可使用。官網有提供一些方法供參考。以簡單效能好來說，我們建議用 Rufus 來製作(教學在後面製作 Acronis 開機碟一併說明)。

下載 Gparted：

  • 下載 Gparted：官網下載頁面
  • 建議：下載最新的 Stable Releases 的 amd64 那個 iso 檔
  • 說明：太新的硬碟可能過舊的 Gparted 讀不到；很舊的硬碟也許最新的 Gparted 不認得，所以要保留一個稍舊版本

上述的說明不是笑話，因為新硬碟的規格舊版本 Gparted 不認識的機率不小，這跟以前一個版本從 XP 時代用到 Win 10 還在服役的情形無法相比。甚至新版(忘了哪一版)無法在記憶體 4G 以下的電腦使用預設模式，所以手上保留一個2、3年前的版本，遇到新硬碟用最新版本就成為目前的解決方式。

 

同場加映：Parted Magic

Parted Magic 是另一個磁碟分割工具，功能較多，實際上是整合很多免費軟體而成的磁碟工具。Parted Magic 對某些主機板或筆電相容性較好，可以做為最佳備胎。

其實 Parted Magic 裡面也包含 Gparted，不過使用 Parted Magic 還有一個最大的理由，那就是可以對 SSD 做 Secure Erase，而且速度很不錯。現在幾乎每台電腦都會使用固態硬碟，雖然 Secure Erase 不需要常做，但是技術員協助學員時使用的機率會高一些，收下來備用吧！

這裡提供的版本不是太新，近來的版本加入了許多我們用不到的東西，體積急速變大，這個版本應該夠用了。

使用 Rufus 製作 USB 隨身碟

Rufus 是一款製作可開機的 USB 磁的工具，開源、免費、無須安裝、體積小、效能高，以製作單一 USB 開機工具來說，Rufus 目前堪稱首選，所以 Gparted、MacriumRescue、Acronis...等都建議使用 Rufus 來製作。

下載 Rufus：

  • 下載：官網下載
  • 說明：建議選最後更新版本的可攜版

 

Rufus 操作範例

下載後的 Rufus 建議用一個資料夾包起來，因為可攜版執行時會產生設定檔案 rufus.ini，如果是一般版本，設定會寫入登錄檔中，功能則完全相同。

執行 Rufus 需要管理員權限，第一次執行會詢問是否檢查更新，選是或否皆可：

 

開啟 Rufus 後如果已經插入 USB 碟會自動找到，建議把其餘 USB 碟(如果有)都暫時退出。介面說明如下：

① → 選取隨身碟 (如果已經插入會自動抓到)
② → 點選要製作的可開機 iso 映像檔
③ → 磁碟標籤，會自動寫入，也可以更改
④ → 都選好之後點執行就開始製作
Ａ → 點這裡可以設置是否檢查更新

Rufus 的其他設置原則上都不必變動，程式會自動根據 iso 檔做出判斷，如果您不清楚其中含意，最好不要變動。

 

出現映像檔偵測後的建議，按照預設點「OK」：

 

出現清除所有在隨身碟上的資料警告，點「確定」：

 

然後 Rufus 就開始轉換 iso 到隨身碟，等到「已就緒」完全變成淺綠色就表示已經完成，點「離開」即可關閉。

在各種製作 USB 開機碟的工具中，我們推薦的 Rufus 從各方面看都是不錯的選擇。至少我們需要的可開機映像檔都能透過 Rufus 製作出來，這對協助學員安裝作業系統的技術員來說有很大的幫助。
