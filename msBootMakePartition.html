使用安裝碟分割磁碟

在還原(安裝)作業系統目前最多的問題是：無法用以前的開機工具，如 AllInOne 的萬用開機碟進入分割磁碟。

這種情形大多出現在舊 USB 開機碟遇到新電腦。因為新的主機板漸漸走向純 UEFI，完全拋棄相容模式(CSM)。這樣就使傳統跑 16 位元的 DOS 工具與許多 32 位元的應用程式都無法載入。

除了使用各種可開機分割工具來處理之外，最簡單又萬用的辦法是使用 Windows 安裝碟進行磁碟分割。

目前 Windows 11 已經全面使用 UEFI + GPT 來安裝，所以用微軟開機碟來分割磁碟是不易出錯又簡單的方式。

製作 Windows 開機碟可以從官網下載(製作)，請參考🚩這裡的教學說明。

不論是 Windows 10 或 11 的開機碟都可以用來分割磁碟，版本不拘。

附錄：將 MBR 磁碟轉成 GPT 格式

上面提到的方式是用於新硬碟或是原本就採用 GPT 格式的磁碟，如果硬碟已經採用 MBR 格式，就必須將其轉成 GPT 格式。

🎈注意：磁碟格式轉換會將裡面的資料全部清除，請把要保留的資料先複製到其他地方。

★操作步驟

  • 同樣使用 Windows 安裝碟開機，跟上面教學一樣一直進行到要選擇分割的地方，然後按「Shift + F10」鍵
  • ❶如下圖出現命令提示字元視窗，輸入「diskpart」按 Enter 鍵
  • ❷然後輸入「list disk」按 Enter 鍵來查看磁碟機編號
  • ❸輸入「select disk x」按 enter 來選擇磁碟 (x 是您要轉換成 GPT 的磁碟編號，此範例是磁碟 0)
  • ❹輸入「clean」按 Enter 鍵把磁碟清空
  • ❺輸入「convert gpt」按 Enter 鍵把磁碟轉換為 GPT 格式
  • 轉換成功，輸入「exit」按 Enter 鍵離開 diskpart
  • 然後關閉此視窗 (按右上角 x 或輸入 exit 離開)

 

這樣就完成了轉換，可以正常分割了。

回到要選擇分割的視窗後，請先點一下「重新整理」，然後就跟上面教學一樣進行分割即可。
